# 技术背景

## 使用技术
- **多端开发框架**: uni-app (基于 Vue 3)
- **UI 框架**: uni-app 内置组件或 uni-ui
- **状态管理**: Pinia (uni-app 推荐)
- **路由管理**: uni-app 内置路由
- **API 交互**: uni.request (uni-app 内置)

## 开发环境
- Node.js (建议 LTS 版本)
- npm 或 yarn (包管理器)
- VS Code (推荐编辑器，配合 uni-app 插件)
- HBuilderX (可选，用于更便捷的 uni-app 开发和调试)

## 技术约束
- 必须是 H5 页面，兼容主流移动浏览器。
- 数据交互必须通过 API 接口进行。
- 考虑前端性能优化，如按需加载、组件懒加载等。
